Gentoo

CLASSIFICAÇÃO

Gentoo Linux é uma metadistribuição que utiliza o sistema de gerenciamento de pacotes Portage. O nome Gentoo é inspirado na espécie de pinguim Gentoo. O gerenciamento de pacotes do Gentoo foi projetado para ser modular, portátil, fácil de manter, flexível e otimizado para a máquina do usuário. Diferentemente da maioria das distribuições de software, os pacotes são geralmente compilados a partir do código-fonte, mantendo a tradição dos ports dos sistemas BSD. No entanto, para conveniência, alguns pacotes grandes são disponibilizados como binários pré-compilados para diversas arquiteturas. O sistema init do Gentoo é uma característica distinta que, embora semelhante ao init do System V usado pela maioria das distribuições Linux, emprega scripts baseados em dependências e níveis de execução nomeados. Utiliza o interpretador runscript para as scripts de inicialização em vez de uma shell tradicional. Originalmente, o sistema rc do Gentoo, escrito em bash e compilado no baselayout 1, enfrentava limitações que exigiam callouts em C, causando inchaço ao longo do tempo. A expansão do Gentoo para plataformas como Gentoo/FreeBSD e Gentoo Embedded dificultou a dependência de um sistema rc baseado em bash, levando ao desenvolvimento do baselayout 2 em C, que requer apenas uma shell compatível com POSIX. Decidiu-se então separar o sistema rc em seu próprio pacote, o OpenRC, desenvolvido principalmente por Roy Marples, que suporta Gentoo e suas variantes, além de outras plataformas como FreeBSD e NetBSD.
