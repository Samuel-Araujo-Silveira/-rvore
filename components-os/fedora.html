Fedora

CLASSIFICAÇÃO

Fedora, anteriormente conhecido como Fedora Core até a versão 7, é um sistema operacional Linux de código aberto e gratuito. Lançado em 2003 e mantido pela comunidade do Projeto Fedora, o Fedora é patrocinado pela Red Hat, que descontinuou seu Red Hat Linux tradicional para focar no Red Hat Enterprise Linux (RHEL). O Fedora serve como base de desenvolvimento para o RHEL. A partir da versão 21, o Fedora foi dividido em três edições principais: Workstation para uso em computadores pessoais, Server para servidores, e Cloud para ambientes de computação em nuvem. Além dessas, existem versões alternativas chamadas "spins" com diversos ambientes gráficos como KDE, Xfce e LXDE, além de edições especializadas para áreas como computação científica, segurança e jogos. O Fedora tem um ciclo de lançamento de aproximadamente seis meses. O nome Fedora se originou de um projeto voluntário associado ao Red Hat Linux, utilizando o logo de um chapéu fedora, iniciado por Warren Togami em 2002. Originalmente, Fedora Core incluía pacotes essenciais mantidos pela Red Hat, enquanto o Fedora Extras, mantido pela comunidade, foi unificado no Fedora 7. O Fedora representa um conjunto de projetos desenvolvidos por uma comunidade global e apoiados pela Red Hat, sempre disponível para uso, modificação e distribuição gratuita (Negus, 2010). O Fedora oferece suporte às principais arquiteturas x86-64 e ARM-hfp. Anteriormente, distribuições especializadas como Pidora e FedBerry atendiam ao Raspberry Pi, mas agora o Fedora ARM abrange o suporte não apenas para o Raspberry Pi, mas também para uma variedade de dispositivos ARM e SBC (computadores de placa única). A partir da versão 26, o Fedora expandiu seu suporte para incluir as arquiteturas ARM AArch64, IBM Power64, IBM Power64le, mipselel, mipsel, IBM Z ("s390x"), MIPS-64el, MIPSel, RISC-V e Intel i686 como arquiteturas secundárias (com i686 sendo primário até a versão 25) (Fedora, 2010).
